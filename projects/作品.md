## 🏆 亮點專案：AI Chat 互動平台

[![AI Chat Banner](https://via.placeholder.com/800x300/1e1e1e/61dafb?text=AI+Chat+Platform+Preview)](https://aichat.tcu.edu.tw)

> 📍 **線上體驗：** [https://aichat.tcu.edu.tw](https://aichat.tcu.edu.tw)  
> 🏷️ **專案定位：** 結合人工智慧技術與現代 Web 前端框架的線上對話平台。

### 🛠️ 技術棧 (Tech Stack)
`React` | `Next.js` | `Python` | `CSS` | `Docker` | `Git`

### 💡 核心功能 (Key Features)

| 功能模組 | 細節說明 |
| :--- | :--- |
| ⚡ **即時互動對話** | 採用 React 與 Next.js 打造高互動性、無延遲的單頁應用程式 (SPA)，具備流暢的 UI/UX。 |
| 🧠 **AI 串接與處理** | 負責將使用者輸入發送至後端 API，妥善處理生成式 AI 的非同步串流回應 (Streaming Responses)。 |
| 🐳 **系統容器化** | 撰寫 `Dockerfile`，將應用程式進行隔離與完整封裝，大幅解決跨伺服器部署的相容性問題。 |

### 🚀 挑戰與突破 (Challenges Overcome)
> **挑戰：** 面對 AI 產生的長篇幅文本，如何保持前端對話列的流暢渲染，並且在正式環境下解決伺服器 CORS (跨域資源共享) 的配置問題？

**解決方案：**
透過查閱官方文件並持續進行效能調校，成功優化了前端元件的渲染頻率。同時，利用 Docker 統一了測試與正式環境的網路層設定，確保前後端 API 溝通順暢且安全。
