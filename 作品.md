## 🏆 亮點專案：AI Chat 互動平台

- **🌐 應用程式連結：** [aichat.tcu.edu.tw](https://aichat.tcu.edu.tw)
- **🛠️ 技術棧：** React, Next.js, Python, CSS, Docker, Git

### 專案簡介 (Project Overview)
這是一個結合人工智慧技術與現代 Web 前端框架的線上對話平台。旨在提供一個無需繁瑣設定，即可在網頁端與強大語言模型進行互動的系統。此專案展示了我在現代網頁開發、API 整合與後端部署上的綜合能力。

### 核心功能 (Key Features)
1. **即時互動對話介面：** 
   - 使用 React 與 Next.js 打造高互動性、無延遲的單頁應用程式（SPA）體驗。
   - 支援對話歷史紀錄顯示與友善的使用者介面設計（UI）。
2. **AI API 串接與資料處理：** 
   - 負責將使用者輸入精確轉換並發送至後端/API。
   - 處理生成式 AI 的文字串流（如果適用）與非同步資料獲取。
3. **系統容器化：** 
   - 透過 Docker 編寫 `Dockerfile` 與環境配置，將應用程式完整封裝，解決跨平台部署的相容性問題。
